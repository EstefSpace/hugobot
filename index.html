Información del Bot

Este bot está diseñado para ayudarte a administrar tu servidor de Discord de manera eficiente y entretener a tus usuarios.

Funciones destacadas:

  • Sistema de bienvenidas con imagen
  • Sistema de despedidas con imagen
  • Comandos muy utiles como /google

¡Únete a nuestro servidor de soporte para obtener ayuda y solicitar nuevas funciones!

Comandos | Checa todo uniendote al servidor de soporte

  • /ayuda: Muestra una lista de comandos disponibles.
  • /util bot: Muestra información sobre el bot.
  • /util user [usuario]: Muestra informacion de un usuario
  • /mod kick [usuario]: Expulsa a un usuario del servidor.
  • /mod ban [usuario]: Prohíbe a un usuario del servidor.
